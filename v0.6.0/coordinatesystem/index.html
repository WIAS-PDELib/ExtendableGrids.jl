Coordinate systems

Coordinate systems are described via abstract types. The list of coordinate systems can be obtained with the coordinatesystems method:

AbstractCoordinateSystem
├─ Cartesian1D
├─ Cartesian2D
├─ Cartesian3D
├─ Cylindrical2D
├─ Cylindrical3D
├─ Polar1D
├─ Polar2D
├─ Spherical1D
└─ Spherical3D
