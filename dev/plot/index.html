ExtendableGrids.PlotterContext — Method
PlotterContext(Plotter; kwargs...)

Initalize plotter context

Plotter: defaults to nothing and can be PyPlot, Plots, VTKView, Makie.

Keyword arguments:

  • colorbar: Show colorbar in plots. Default: true

  • aspect: Aspect ration modification. Default: 1.0

  • clear: Clear plot before new plot.. Default: true

  • subplot: Subplot number for multiple plots. Default: 111

  • colormap: Contour plot colormap. Default: hot

  • legend: Add legend to plot. Default: true

  • label: Label of plot. Default: f

  • elevation_factor: Height factor for elevation. Default: 1.0

  • elevation: Show elevation plot (2D function graph). Default: false

  • layout: Layout of multiple plots (experimental). Default: (1, 1)

  • axisgrid: Show background grid in plots. Default: true

  • resolution: Plot resolution for pairs. Default: (300, 300)

  • color: Color of lines on plot. Default: (0, 0, 0)

  • colorlevels: Number of color levels for contour plot. Default: 51

  • isolines: Number of isolines in contour plot. Default: 11

  • show: Show plots immediately. Default: true

  • legend_location: Location of legend. Default: upper right

  • fignumber: Figure number (PyPlot). Default: 1

  • framepos: Subplot position in frame (VTKView). Default: 1
