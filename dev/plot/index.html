Plotting

Provide plotting facilities for various Julia graphics APIs. A graphics API (Plots, PyPlot, VTKView) needs to be passed to plot in the Plotter keyword parameter

ExtendableGrids.plot — Method
plot(grid::ExtendableGrid, U::AbstractArray{T,1} where T; Plotter, color, cmap, label, levels, aspect, clear, show, p)

Plot lowest order continuous finite element function on grid defined by values on the grid nodes.

Keyword arguments:

  • Plotter: defaults to nothing and can be PyPLot, Plots, VTKView.
  • color: color of plot on 1D grid
  • cmap: color map for heatmap plot
  • label: label of plot
  • levels: number of isolevels
  • aspect: aspect ratio of plot
  • clear: if true (default) clear plot before plotting
  • show: if true (default) show plot
